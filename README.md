Допустимые значения атрибута Должность: медсестра
- няня
- воспитательница
- подсобный рабочий
- начальство.
Допустимые значения атрибута Дата_рождения больше 01.01.1900.
Допустимые значения атрибута Пол:
- мужской
- женский.
